## List Products

`GET /api/v1/products`

Returns a paginated list of products with optional filtering

### Query Parameters

- **page** (integer, default: 1): Page number for pagination
- **per_page** (integer, default: 15): Number of items per page
- **search** (string): Search term to filter products
- **category** (string): Category name to filter products

### Responses

- **200 OK**: Successful operation

---

## Search Products

`GET /api/v1/products/search`

Search products by term and return paginated results

### Query Parameters

- **term** (required, string): Search term to find products
- **page** (integer, default: 1): Page number for pagination
- **per_page** (integer, default: 15): Number of items per page

### Responses

- **200 OK**: Successful operation

---

## Get Product Details

`GET /api/v1/products/{product}`

Returns details for a single product

### Path Parameters

- **product** (required, integer): Product ID

### Response

Returns a product resource with the following information:

```json
{
  "id": 1,
  "name": "Product Name",

  "variant": {
    "id": 1,
    "name": "Standard Edition",
```
