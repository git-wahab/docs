## Create Direct Purchase Order

`POST /api/v1/orders`

Creates an order directly without going through the cart flow.

### Request Body

```json
{
  "wallet_id": "123e4567-e89b-12d3-a456-426614174002",
  "items": [
    {
      "product_variant_id": 123,
      "quantity": 1
    }
  ]
}
```

### Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `wallet_id` | string (UUID) | ✅ | ID of the wallet to use for payment |
| `items` | array | ✅ | Array of items to purchase |
| `items[].product_variant_id` | integer | ✅ | ID of the product variant |
| `items[].quantity` | integer | ✅ | Quantity to purchase (minimum: 1) |

### Responses

| Status | Description |
|---|---|
| `201 Created` | Order created successfully |
| `401 Unauthorized` | Missing or invalid authentication token |
| `403 Forbidden` | Access denied |
| `422 Unprocessable Entity` | Validation error or business rule violation (e.g., "Insufficient funds in wallet") |
| `500 Internal Server Error` | Server-side error |

### 422 — Validation / Business Rule Error

```json
{
  "message": "Insufficient funds in wallet"
}
```

### 500 — Server Error

```json
{
  "message": "An unexpected error occurred while processing your order."
}
```
