## List Orders

`GET /api/v1/orders`

Returns a paginated list of orders for the authenticated user with optional filtering

### Query Parameters

- **per_page** (integer, default: 15, max: 100): Number of items per page
- **fulfillment_status** (string): Filter by fulfillment status
  - Options: `pending`, `processing`, `completed`, `refunded`
- **payment_status** (string): Filter by payment status
  - Options: `pending`, `completed`

### Responses

- **200 OK**: Successful operation
- **401 Unauthorized**: Missing or invalid authentication token
- **403 Forbidden**: Access denied

## Create Direct Purchase Order

`POST /api/v1/orders`

Creates an order directly without going through the cart flow

### Request Body

The request should include:

```json
{
  "wallet_id": "123e4567-e89b-12d3-a456-426614174002",
  "items": [
    {
      "product_variant_id": 123,
      "quantity": 1
    }
  ]
}
```

### Parameters

- **wallet_id** (required, string): ID of the wallet to use for payment
- **items** (required, array): Array of items to purchase
  - **product_variant_id** (required, integer): ID of the product variant
  - **quantity** (required, integer): Quantity to purchase (minimum: 1)

### Responses

- **201 Created**: Order created successfully
- **401 Unauthorized**: Missing or invalid authentication token
- **403 Forbidden**: Access denied
- **422 Unprocessable Entity**: Validation error or business rule violation (e.g., "Insufficient funds in wallet")
- **500 Internal Server Error**: Server-side error

### Example Response

```json
{
  "message": "Order created successfully",
  "data": {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "order_number": "#ORD-12345",
    "fulfillment_status": "pending",
    "payment_status": "completed",
    "subtotal": {
      "amount": 99.99,
      "formatted": "€99.99"
    },
    "conversion_fees": {
      "amount": 2.5,
      "formatted": "€2.50"
    },
    "total": {
      "amount": 102.49,
      "formatted": "€102.49"
    },
    "currency": "EUR",
    "items": []
  }
}
```

## Get Order Details

`GET /api/v1/orders/{order}`

Returns detailed information about a specific order

### Path Parameters

- **order** (required, string/UUID): Order ID

### Responses

- **200 OK**: Successful operation
- **401 Unauthorized**: Missing or invalid authentication token
- **403 Forbidden**: Access denied
- **404 Not Found**: Order not found

---

## Download Order Digital Codes

`GET /api/v1/orders/{order}/download-codes`

Downloads a ZIP file containing digital codes for the order

### Path Parameters

- **order** (required, string/UUID): Order ID

### Responses

- **200 OK**: ZIP file download
- **401 Unauthorized**: Missing or invalid authentication token
- **403 Forbidden**: Access denied
- **404 Not Found**: Order not found or no codes available

---

## Get Order Digital Codes (JSON)

`GET /api/v1/orders/{order}/get-codes`

Returns digital codes for the order in JSON format

### Path Parameters

- **order** (required, string/UUID): Order ID

### Response Format

```json
{
  "error": false,
  "message": "Codes retrieved successfully",
  "data": [
    {
      "code": "XXXX-XXXX-XXXX-XXXX",
      "product_name": "Product Name"
    }
  ]
}
```

### Responses

- **200 OK**: Successful operation
- **401 Unauthorized**: Missing or invalid authentication token
- **403 Forbidden**: Access denied
- **404 Not Found**: Order not found or no codes available
