## List Orders

<OpenAPI method="get" path="/orders">
  Returns a paginated list of orders for the authenticated user with optional filtering
</OpenAPI>

### Query Parameters

- **per_page** (integer, default: 15, max: 100): Number of items per page
- **fulfillment_status** (string): Filter by fulfillment status
  - Options: `pending`, `processing`, `completed`, `refunded`
- **payment_status** (string): Filter by payment status
  - Options: `pending`, `completed`

### Responses

- **200 OK**: Successful operation
- **401 Unauthorized**: Missing or invalid authentication token
- **403 Forbidden**: Access denied

## Download Order Digital Codes

<OpenAPI method="get" path="/orders/{order}/download-codes">
  Downloads a ZIP file containing digital codes for the order
</OpenAPI>

### Path Parameters

- **order** (required, string/UUID): Order ID

### Responses

- **200 OK**: ZIP file download
- **401 Unauthorized**: Missing or invalid authentication token
- **403 Forbidden**: Access denied
- **404 Not Found**: Order not found or no codes available

## Get Order Digital Codes (JSON)

<OpenAPI method="get" path="/orders/{order}/get-codes">
  Returns digital codes for the order in JSON format
</OpenAPI>

### Path Parameters

- **order** (required, string/UUID): Order ID

### Response Format

```json
{
  "error": false,
  "message": "Codes retrieved successfully",
  "data": [
    {
      "code": "XXXX-XXXX-XXXX-XXXX",
      "product_name": "Product Name"
    }
  ]
}
```

### Responses

- **200 OK**: Successful operation
- **401 Unauthorized**: Missing or invalid authentication token
- **403 Forbidden**: Access denied
- **404 Not Found**: Order not found or no codes available

## List Products

<OpenAPI method="get" path="/products">
  Returns a paginated list of products with optional filtering
</OpenAPI>

### Query Parameters

- **page** (integer, default: 1): Page number for pagination
- **per_page** (integer, default: 15): Number of items per page
- **search** (string): Search term to filter products
- **category** (string): Category name to filter products

### Responses

- **200 OK**: Successful operation
